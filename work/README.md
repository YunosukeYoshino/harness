# Work artifacts

`work/` は mutable artifact の置き場です。

- `specs/`: spec-first prompt / spec
- `plans/`: 実装計画
- `sprint-contracts/`: scope / done 定義
- `reviews/`: review artifact
- `qa-reports/`: runtime / UI QA
- `progress/`: 進捗
- `handoffs/`: 次セッションへの引き継ぎ

公開 template では空ディレクトリだけを version control に残します。
