# Database options

## Prisma

- ORM と migration をまとめて扱うなら Prisma
- local template では `packages/integrations/prisma/schema.prisma` を入口にする

## Supabase

- hosted Postgres + auth + storage が必要なら Supabase
- skills を入れて schema / RLS / migration の指示精度を上げる
