# Documentation tiers

- `AGENTS.md` / `CLAUDE.md`: 短い入口
- `docs/`: durable knowledge
- `work/`: mutable artifact

## Do not do

- root に `PLAN.md`, `ARCHITECTURE_NOTES.md`, `QA.md` を増やす
- code change と無関係な長文を `AGENTS.md` に積む
- `work/` の進捗を root README に書く
